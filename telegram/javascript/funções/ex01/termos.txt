* Neste exercício, vamos criar um programa de bordo para nossa nave que receba o nome da nave.
* A velocidade da nave deve ser inicialmente 0 e um menu deve ser exibido para que o usuário escolha entre as seguintes opções:
  1- Acelerar a nave em 5km/s
  2- Desacelerar em 5km/s // A nave não pode ter um velocidade menor que 0km/s
  3- Imprimir dados de bordo // Deve ser impresso o nome da nave com sua velocidade atual
  4- Sair do programa
* Caso a pessoa escolha um número que não está na lista, é necessário exibir o menu novamente até que seja escolhida uma opção válida. Para fazer este app, precisaremos:
  1- Criar uma função que exiba o menu e valide a opção escolhida.
  2- Uma função para cada ação do menu de acelerar.
  3- Encerrar o app somente quando digitar sair
